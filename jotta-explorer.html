Riprendi il controllo del tuo spazio Jottacloud

Jotta Explorer è un’applicazione desktop per Windows progettata per esplorare, cercare e organizzare con maggiore facilità grandi quantità di dati archiviate su Jottacloud.

Quando un account contiene centinaia di migliaia di file distribuiti tra Archive, Sync, Backup e gli altri mount point disponibili, orientarsi attraverso il client ufficiale o l’interfaccia web può diventare difficile. Jotta Explorer offre un’esperienza più vicina a Esplora file di Windows e permette di scegliere se cercare ricorsivamente o limitare la ricerca alla cartella corrente.

Funzioni principali

  • navigazione live nei mount point disponibili;
  • indice locale per ricerche rapide;
  • ricerca nel mount point o nella cartella corrente;
  • filtri per nome, percorso, estensione e tipo;
  • download di file e cartelle con coda;
  • cache locale, anteprime, preferiti e recenti;
  • spostamento tra Archive e Sync;
  • navigazione disponibile anche durante la creazione dell’indice.

Un solo token

È sufficiente generare un token Jottacloud e inserirlo nell’applicazione. Credenziali, indice, cache e impostazioni rimangono nella cartella dati locale.

Anteprime

Le anteprime possono essere generate dopo il download del file nella cache locale. I file non ancora scaricati restano comunque navigabili e ricercabili.

Cosa non fa

Jotta Explorer non sostituisce il client ufficiale, non mantiene cartelle sincronizzate e non esegue backup automatici. È uno strumento complementare dedicato a navigazione, ricerca, download e organizzazione.

Jotta Explorer è un progetto indipendente e non è affiliato, sponsorizzato o approvato da Jottacloud.

Take back control of your Jottacloud storage

Jotta Explorer is a Windows desktop application designed to make it easier to browse, search and organize large amounts of data stored on Jottacloud.

When an account contains hundreds of thousands of files across Archive, Sync, Backup and other mount points, the official client and web interface can become difficult to use. Jotta Explorer provides an experience closer to Windows File Explorer and lets you choose between recursive search and current-folder search.

Main features

  • live browsing across available mount points;
  • local indexing for fast search;
  • search across the mount point or current folder;
  • filters by name, path, extension and type;
  • file and folder downloads with queue management;
  • local cache, previews, favorites and recent items;
  • moving content between Archive and Sync;
  • cloud browsing while the index is being built.

One token

Generate one Jottacloud token and enter it in the application. Credentials, indexes, cached files and settings remain in the local application data folder.

Previews

Previews can be generated after a file has been downloaded to the local cache. Files not yet downloaded can still be browsed and found.

What it does not do

Jotta Explorer does not replace the official client, maintain synchronized local folders or perform automatic backups. It is a complementary tool focused on browsing, search, download and organization.

Jotta Explorer is an independent project and is not affiliated with, sponsored by or endorsed by Jottacloud.
