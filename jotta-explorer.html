Perché Jotta Explorer

Why Jotta Explorer

Quando un account contiene centinaia di migliaia di file distribuiti tra Archive, Sync, Backup e altri mount point, orientarsi tramite il client ufficiale o l’interfaccia web può diventare difficile. Jotta Explorer offre un’esperienza più vicina a Esplora file di Windows.

When an account contains hundreds of thousands of files across Archive, Sync, Backup and other mount points, the official client and web interface can become difficult to use. Jotta Explorer provides an experience closer to Windows File Explorer.

Funzioni principali

Main features

  • navigazione live nei mount point disponibili;
  • indice locale per ricerche rapide;
  • ricerca nel mount point o nella cartella corrente;
  • filtri per nome, percorso, estensione e tipo;
  • download di file e cartelle con coda;
  • cache locale, anteprime, preferiti e recenti;
  • spostamento tra Archive e Sync.
  • live browsing across available mount points;
  • local indexing for fast search;
  • mount-point or current-folder search;
  • filters by name, path, extension and type;
  • file and folder downloads with queue management;
  • local cache, previews, favourites and recent items;
  • moving content between Archive and Sync.

Un solo token

One token

È sufficiente generare un token Jottacloud e inserirlo nell’applicazione. Credenziali, indice, cache e impostazioni rimangono nella cartella dati locale.

Generate one Jottacloud token and enter it in the application. Credentials, indexes, cached files and settings remain in the local application data folder.

Anteprime e cache

Previews and cache

Le anteprime possono essere generate dopo il download del file nella cache locale. I file non ancora scaricati restano comunque navigabili e ricercabili.

Previews can be generated after a file has been downloaded to the local cache. Files not yet downloaded can still be browsed and found.

Cosa non fa

What it does not do

Jotta Explorer non sostituisce il client ufficiale, non mantiene cartelle sincronizzate e non esegue backup automatici. È uno strumento complementare dedicato a navigazione, ricerca, download e organizzazione.

Jotta Explorer does not replace the official client, maintain synchronized local folders or perform automatic backups. It is a complementary tool focused on browsing, search, download and organization.
