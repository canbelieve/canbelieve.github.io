Riprendi il controllo del tuo spazio Jottacloud

Jotta Explorer è un'applicazione desktop progettata per esplorare, cercare, scaricare e organizzare con maggiore facilità grandi quantità di dati archiviate su Jottacloud.

Jotta Explorer — Italiano

Il progetto nasce da un'esigenza concreta: quando un account contiene centinaia di migliaia di file distribuiti tra Archive, Sync, Backup e gli altri mount point disponibili, orientarsi attraverso il client ufficiale o l'interfaccia web può diventare difficile.

La ricerca offerta dagli strumenti ufficiali opera generalmente sull'intero spazio cloud e non sempre permette di circoscrivere in modo efficace i risultati alla cartella che si sta consultando. Con archivi molto grandi, questo può produrre troppi risultati e rendere complicate le normali attività di ricerca, manutenzione e riordino.

Jotta Explorer affronta il problema offrendo un'esperienza più vicina a Esplora file di Windows: permette di navigare all'interno dei mount point Jottacloud, entrare nelle cartelle e scegliere se effettuare una ricerca ricorsiva oppure limitarla alla posizione corrente.

Funzioni principali

Navigazione live

Esplora i mount point disponibili nell'account Jottacloud anche prima della creazione dell'indice.

Indice locale

Crea un indice salvato sul computer per ottenere ricerche molto più rapide e precise.

Ricerca mirata

Cerca nell'intero mount point selezionato oppure soltanto nella cartella corrente, anche con filtri avanzati.

Download e cache

Scarica file e cartelle, controlla la coda e conserva localmente i contenuti già recuperati.

Organizzazione

Usa preferiti, elementi recenti, anteprime e strumenti per spostare contenuti tra Archive e Sync.

Integrazione Windows

Apri i file con le applicazioni associate e visualizza cartelle e risultati con un ordinamento familiare.

Un solo token per accedere all'account

La configurazione è semplice: è sufficiente generare un token Jottacloud e inserirlo nell'applicazione.

Jotta Explorer utilizza il token per individuare automaticamente le risorse disponibili e permettere l'accesso ai dati presenti nei diversi mount point dell'account, senza dover configurare separatamente ogni posizione.

Credenziali, indice, cache e impostazioni vengono mantenuti localmente nella cartella dati dell'applicazione.

Anteprime e miniature

Jottacloud non rende sempre disponibili miniature e anteprime direttamente attraverso l'accesso utilizzato dall'applicazione. Jotta Explorer può quindi generare l'anteprima di un file soltanto dopo averlo scaricato nella cache locale del computer.

I file non ancora scaricati possono comunque essere esplorati e trovati, ma la loro miniatura potrebbe non essere disponibile.

Cosa Jotta Explorer non fa

Jotta Explorer non sostituisce il client ufficiale Jottacloud e non esegue la sincronizzazione automatica dei file.

  • non mantiene una cartella locale sincronizzata con il cloud;
  • non effettua backup automatici;
  • non replica il funzionamento del client ufficiale;
  • non modifica il sistema di sincronizzazione di Jottacloud.

È uno strumento complementare dedicato alla navigazione, alla ricerca, al download e all'organizzazione mirata dei contenuti già presenti nel proprio account.

Perché usare Jotta Explorer?

Chi utilizza Jottacloud dispone spesso di una quantità enorme di dati. Con il passare del tempo, trovare un file, ricostruire la struttura delle cartelle o riordinare l'archivio può diventare molto difficile.

Jotta Explorer è stato creato proprio per questo: offrire una visione più chiara e controllabile del proprio cloud e rendere nuovamente semplici operazioni come cercare, esplorare, scaricare e organizzare i file.

In breve, Jotta Explorer aiuta a riprendere il controllo del proprio spazio Jottacloud.

Jotta Explorer — English

Take back control of your Jottacloud storage.

Jotta Explorer is a Windows desktop application designed to make it easier to browse, search, download and organize large amounts of data stored on Jottacloud.

It provides an experience closer to Windows File Explorer, with live browsing across available mount points, a local search index, folder-scoped searches, advanced filters, downloads, local cache, previews, favorites and recent items.

The application uses a single Jottacloud token to discover the available resources automatically. Credentials, indexes, cached files and settings are stored locally.

Jotta Explorer does not replace the official Jottacloud client, does not maintain synchronized local folders and does not perform automatic backups. It is a complementary tool focused on browsing, searching, downloading and organizing content already stored in the account.
